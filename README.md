## Описание
**Light Path** — это логическая головоломка, где игроку необходимо настроить оптическую систему зеркал и фильтров, чтобы направить луч света от источника к приёмнику.

## Геймплей
1.  На поле расположен **Источник света** и один или несколько **Приёмников**.
2.  Игрок расставляет **Зеркала** на сетке.
3.  **Клик мыши** по зеркалу поворачивает его на 90°.
4.  Луч света пересчитывается в реальном времени.
5.  **Цель:** Луч должен достичь приёмника. В продвинутых уровнях цвет луча должен совпадать с цветом приёмника (используются фильтры).

### Основной функционал (MVP)

1.  **Игровое поле:** Сетка 10×10 клеток, отрисовка через `Graphics`.
2.  **Базовые объекты:** Источник света, Приёмник, Зеркала (отражение на 90°).
3.  **Логика луча:** Алгоритм пошагового распространения луча.
4.  **Интерактивность:** Поворот зеркал по клику мыши в реальном времени.
5.  **Условие победы:** Проверка попадания луча в приёмник.
6.  **Система уровней:** 5 встроенных уровней разной сложности.
7.  **Интерфейс:** Кнопки управления (Сброс, Далее, Выход).

### Перспективный функционал 

1.  **Цветовые фильтры:** Изменение цвета луча при прохождении через препятствие.
2.  **Цветные приёмники:** Требуют попадания луча определённого спектра.
3.  **Стены и препятствия:** Блокируют прохождение луча.
4.  **Счётчик ходов:** Подсчёт количества поворотов зеркал за уровень.
5.  **Таймер:** Отсчёт времени прохождения уровня.
6.  **Звуковые эффекты:** Сопровождение действий звуками (через `System.Media`).
7.  **Сохранение прогресса:** Запись пройденных уровней в файл.

**Результат:** запускаешь — видишь тёмную пустую сетку.

### Неделя 2 ✅ — Объекты на поле

- Каждая клетка теперь знает, что на ней стоит — пусто, источник или зеркало
- На поле появился источник света — жёлтый круг
- На поле появились зеркала — прямоугольники с линией внутри
- Линия внутри зеркала показывает его угол — либо `\` либо `/`
- Зеркало можно кликнуть мышью — оно поворачивается на 90 градусов
- При наведении мыши на зеркало курсор меняется на руку
- Написали автоматические тесты — 11 проверок что всё работает правильно
- Тесты показывают результат во всплывающем окне при запуске

**Результат:** запускаешь — видишь источник и зеркала. Кликаешь по зеркалу — оно поворачивается.

---

### Неделя 3 — Луч света

- Из источника выходит жёлтая линия — луч света
- Луч идёт в одном направлении по клеткам
- Когда луч доходит до зеркала — он отражается и меняет направление
- Зеркало `\` отражает луч в одну сторону
- Зеркало `/` отражает луч в другую сторону
- Если луч доходит до края поля — он исчезает
- Клетки через которые прошёл луч — слегка подсвечиваются
- Каждый раз когда поворачиваешь зеркало — луч мгновенно пересчитывается
- Написали тесты — проверяем что луч правильно отражается и останавливается

**Результат:** запускаешь — видишь жёлтую линию от источника. Крутишь зеркала — линия меняет направление.

---

### Неделя 4 — Приёмник, победа и уровни

- На поле появился приёмник — зелёный квадрат с мишенью внутри
- Цель игры — направить луч в этот приёмник
- Когда луч попадает в приёмник — он загорается ярче
- Появляется надпись "Уровень пройден!"
- Добавили кнопку "Сброс" — возвращает зеркала в начальное положение
- Добавили кнопку "Следующий уровень" — переход к новой головоломке
- Кнопка следующего уровня заблокирована пока не решишь текущий
- Сверху показывается номер текущего уровня — например "Уровень 1 из 3"
- Сделали три разных уровня с разным расположением зеркал
- Каждый следующий уровень сложнее предыдущего
- Написали тесты — проверяем победу, сброс и переключение уровней

**Результат:** полноценная играбельная игра с тремя уровнями.

| **Следующий уровень** | Кнопка «Next» (доступна после победы) |
